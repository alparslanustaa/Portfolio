### Frontend - Running React Client

1. Enter frontend subdirectory `cd portfolio-react`
2. Install dependencies with `npm install`.
3. Start client `npm start`

## Dependencies

- Node 14.x or above
- NPM 5.x or above
- postcss": ^8.4.14"
- tailwindcss: ^3.0.24
